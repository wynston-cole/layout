Floats

The layout of Multicolumn

Multicolumn can be split into multiple columns to help reduce long list.like If you have a really long list of elements, such as a list of all of the countries of the world, it can result in a lot of scrolling and time wasted for a user.

Positioning

The position property changes how an element behaves in the normal flow of the document, and how it relates to other elements.Adding position: relative to an element also makes it the containing block of any child elements with position: absolute.An element with a position value of fixed behaves in a similar way to absolute, with its parent being the root element.
